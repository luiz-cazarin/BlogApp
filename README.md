# BlogApp
 Blog in nodejs

# Foi utilizado
<p>
 Nodejs<br>
 MongoDB - Banco de dados noSQL<br>
 Handlebars -  Processador de templates que gera a tela HTML dinamicamente<br> 
 Express.js - framework para aplicações web para Nodejs<br>
 
</p>

## Para subir o projeto localmente
npm node.js
